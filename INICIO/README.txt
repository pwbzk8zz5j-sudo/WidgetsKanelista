KANELISTA WIDGETS

Archivos:
- home.html: widget principal.
- style.css: colores, tamaños, fondo y chispitas.
- quotes.js: frases.
- script.js: hora, fecha, saludo y lógica.

Para probarlo:
1. Abre home.html en tu navegador.

Para publicarlo:
1. Sube esta carpeta completa a GitHub.
2. Activa GitHub Pages o publícala en Netlify.
3. Usa la URL pública de home.html dentro de un bloque /embed en Notion.

Importante:
No subas solo home.html. Los cuatro archivos deben permanecer juntos.
